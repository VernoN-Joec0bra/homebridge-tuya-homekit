# BNETA Smart Bulb → Apple HomeKit via Homebridge

Connects a Tuya-based smart bulb to Apple HomeKit using Homebridge on macOS.

## Requirements
- Node.js (via NVM)
- Homebridge
- homebridge-tuya-platform
- Tuya Developer account

## Setup
1. Clone this repo
2. Copy config.template.json to config.json
3. Fill in your real credentials in config.json
4. Run: homebridge

## Important
Never commit your real config.json — it contains API secrets.
